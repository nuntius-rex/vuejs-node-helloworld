## Setup Option 1
```
git clone https://github.com/nuntius-rex/vuejs-node-helloworld.git

npm install
```

## Setup Option 2
You can also create this example with the Vue.js CLI yourself.

```
npm install -g @vue/cli
vue create hello-world
npm install
```
